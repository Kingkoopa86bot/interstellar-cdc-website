# Interstellar CDC Website

## Deploy on Vercel
1. Create or log into Vercel.
2. Create a new project.
3. Upload or import this folder.
4. Click Deploy.
5. Go to Project > Settings > Domains.
6. Add your domain.
7. Copy the DNS records Vercel provides into your domain DNS settings.

## Local preview
npm install
npm run dev

## Build
npm run build
